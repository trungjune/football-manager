Available Endpoints

GET /api/status

Returns the current status of the API.

Example Response:

{
  "status": "ok",
  "message": "API is working!",
  "timestamp": "2025-05-30T08:32:50.123Z",
  "version": "1.0.0"
}

GET /api/v1/status

Returns the current status of the API (v1 version).

Example Response:

{
  "status": "ok",
  "message": "API is working!",
  "timestamp": "2025-05-30T08:32:50.123Z",
  "version": "1.0.0"
}

GET /api/health

Health check endpoint for the backend API.

Example Response:

{
  "status": "ok",
  "message": "API is working!",
  "timestamp": "2025-05-30T08:32:50.123Z",
  "env": "production",
  "version": "1.0.0"
}

GET /api/public

Public endpoint that returns information about available API endpoints.

Example Response:

{
  "status": "ok",
  "message": "Public API is working!",
  "timestamp": "2025-05-30T08:32:50.123Z",
  "endpoints": [
    {
      "path": "/api/public",
      "method": "GET",
      "description": "This endpoint"
    },
    {
      "path": "/api/health",
      "method": "GET",
      "description": "Health check endpoint"
    }
  ]
}
